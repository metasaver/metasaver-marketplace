## Field Descriptions

- [file-path]: Relative path from repository root
- [line-number]: Starting line of the signature
- [signature]: Function/type/component name with basic signature
- [package]: One of: utils, contracts, ui-components
- [module]: Path within package (e.g., string-helpers, types/user)
- [export-name]: Name of the exported item to import
- [percentage]: Similarity score 0-100%
- [comparison]: Brief explanation of what matches

## Formatting Guidelines

- Use relative paths from repo root for file references
- Keep comparisons to 1 sentence
- Show exact import statement to fix violation
- Group violations by severity (‚ùå ‚Üí ‚ö†Ô∏è ‚Üí ‚ÑπÔ∏è)
- Always include summary footer if any findings exist
